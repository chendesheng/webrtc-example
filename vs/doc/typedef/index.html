    public ChatWindow: Object source

    Properties:

    NameTypeAttributeDescription
    minimize function(): Promise<void, Error>

    Minimize current chat window
    close function(): Promise<void, Error>

    Close current chat window
    endChat function(): Promise<void, Error>

    End current chat
    on function(type: string, callback: function): ChatWindow | function(visitor: ChatWindowOnVisitor): ChatWindow

    Register callback for specific events. Accepted events are:

    • minimize - Trigger when chat window is minimized
    • close - Trigger when chat window is closed
    • endchat - Trigger when chat is ended

    In short, function can also provide a visitor object containing key-value pairs, where key is the type string and value is the callback function for that specific type.
    render function(type: string, callback: RenderCallback): ChatWindow | function(visitor: RenderCallbackVisitor): ChatWindow

    Render customized element. If callback provies an HTMLElement, original component will not be rendered; if provides null, use original component instead.

    Function also accepts a visitor for multiple callback registration.

    Possible render types are:

    • 'prechat.header'
    • 'prechat.content'
    • 'prechat.footer'
    • 'chat.header'
    • 'chat.content'
    • 'chat.input'
    • 'chat.footer'
    • 'postchat.header'
    • 'postchat.content'
    • 'postchat.footer'
    • 'offline.header'
    • 'offline.content'
    • 'offline.footer'
    • '*.header'
    • '*.content'
    • '*.footer'

    public MessageTransformVisitor: Object source

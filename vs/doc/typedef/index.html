    public Field: Object source

    Properties:

    NameTypeAttributeDescription
    type string

    Type of field, possible types are:

    • 'input' - single line input field
    • 'textarea' - Multiple line input field
    • 'radiolist' - Group of radio buttons
    • 'checkbox' - Single checkbox field
    • 'select' - Dropdown list field
    • 'checkboxlist' - Group of checkbox
    • 'file' - Attachment upload field
    • 'email' - Input field that accept email address
    • 'ticket' - Input field that accept Comm100 Ticket number
    • 'department' - Dropdown list that display available department
    • 'stars' - Field for user to rate the chat (select from 1 to 5 stars by default)
    • 'custom' - Custom field that display whatever user provide in element property
    name string

    Label name of field
    value string | number

    Value of current field
    isSystem boolean

    Whether the field is system field
    error string

    Potential error of this field with current value. This error might or might not be displayed, usually it will be displayed after field blur or before form submit.
    options Array<{type: string, name: string}>

    Possible options of current field, only valid when type is select, radiobox or checkbox.
    element HTMLElement

    Extended element, only valid when type is custom
